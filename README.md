# 任務管理系統

這是一個使用 Flask、Jinja2 與 SQLite 製作的任務管理 Web 應用程式。

## 系統功能

- 新增任務
- 查看任務列表
- 編輯任務
- 刪除任務
- 標記任務完成或未完成
- 依照完成狀態篩選任務

## 技術棧

- Flask
- Jinja2
- SQLite
- HTML / CSS
